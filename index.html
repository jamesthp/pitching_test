Beer Run

Some info about Beer Run

Beer Run is designed to bring coordination and automation that the most important of Friday afternoon activities... the beer run.
One person is nominated at random as the 'runner' and will receive an automated notification at the start time. A points system will track speed, quantity of a single carry and quality of selection.
At the end of each month the 'runner' with the highest tally will win the grand prize of... beer! And one month off beer run duties.

ACES Colour

Some info about ACES Colour

ACES is an industry standard VFX colour workflow. It standardises the working colour space for all VFX tools across all departments. Adopting this workflow globally will enable seamless and consistent colour operations across sites and simplify the sharing of VFX projects.

Auto USD

Some info about Auto USD

Auto USD is an automated scene assembly tool built on the Universal Scene Description technology from Pixar. This tool enables the automated build, layout, lighting and render submission of a sequence of shots.
